# moonclown

Данный проект сделан в качестве отборочного задания на стажировку в компанию Lad! Проект является доработкой моего проекта movies-explorer, который был сделан мною на курсах от яндекса по веб-разработке. Возможно, вы уже видели у кого-то похожий проект. Единственная схожесть со старым проектом заключается в основном функционале приложения, заключающимся в работе с массивом фильмов, но в проекте множество нововведений:

- Аутентификация почты по коду
- .css -> .module.scss
- Fetch -> Axios
- +Redux Toolkit
- Дополнительные доработки (новые роуты и контроллеры на бэкенде, возможность изменения пароля, новая функциональность в компоненте для несуществующих роутов, полностью изменены стили)
- Исправлено множество ошибок, оптимизирован код

В данном проекте есть 7 страниц:

1. Главная страница. Доступна по роуту '/'. На данную страницу может зайти как авторизованный, так и неавторизованный пользователь.

2. Страница с фильмами. Доступна по роуту '/movies'. На этой странице можно искать фильмы среди тех, что приходят с API (всего 100 фильмов). Также можно переходить на ютуб-трейлер данного фильма (многие уже удалены или заблокированы, так как данные достаточно устаревшие), можно лайкать фильмы и таким образом сохранять их на роут '/saved-movies'.

3. Страница с сохранёнными фильмами. Доступна по роуту '/saved-movies'. На данной странице отображаются только те фильмы, которые вы лайкнули. По нажатию на крестик их можно удалить с данной страницы.

4. Страница профиля. Доступна по роуту '/profile'. На данной странице вы можете менять данные своего профиля, а также выйти из текущего аккаунта.

5. Страница регистрации. Доступна по роуту '/signup'. На данной странице вы можете зарегистрироваться. При успешном подтверждении кода, пришедшего на почту, ваши данные сохранятся в базу, сразу же произойдёт логин и направление на страницу с фильмами.

6. Страница логина. Доступна по роуту '/signin'. На данной странице можно авторизоваться, если у вас уже есть аккаунт. При успешной авторизации вас направит на страницу с фильмами.

7. Страница с ошибкой 404. Доступна по любому роуту, который не предусмотрен в приложении. При переходе на данную страницу, пользователь может нажать кнопку "назад", которая вернёт его на главную страницу проекта, а также в данном компоненте находится загадочная свинка, которую лучше не злить =`)`

Проект использует 2 api:

1. 'https://api.films.nasdermn.nomoreparties.sbs/'; (моё апи с основным функционалом)
2. 'https://api.nomoreparties.co/beatfilm-movies'; (апи, с которого я лишь получаю массив из 100 фильмов)

Я задеплоил проект на сервере Яндекс.Облака. Его можно найти по следующему домену:

https://films.nasdermn.nomoreparties.sbs/

На данный момент в проекте есть 1 проблема: если использовать при регистрации автозаполнение, предлагаемое яндексом, и нажать кнопку "получить код" при регистрации, то инпут с почтой очистится из-за глупого поведения яндекс форм (инпут переходит в состояние disabled и видимо это вызывает его очистку). Данная проблема присутствует только если использовать автозаполнение яндекса, без него проект функционирует замечательно. Данная проблема будет решена в ближайшее время, а пока, если вы использовали автозаполнение, просто обновите страничку и напишите имя и пароль заново - почта все равно сохраняется в локальном хранилище и подгружается оттуда.
